class romidata.task.RomiTask(*args, **kwargs)¶

Implementation of a luigi Task for the romidata DB API.

complete()¶

Checks if a task is complete by checking if Filesets corresponding to te task id exist.

Contrary to original luigi Tasks, this check for completion of all required tasks to decide wether it is complete.

output()¶

Output for a RomiTask is a FileSetTarget, the fileset ID being the task ID.
